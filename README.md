# Aditya's Portfolio

A basic portfolio app built with Next.js to showcase personal interests and lifestyle.

## Planned Components

- **Things I Own** - A showcase of my favorite possessions
- **Desk Setup** - My workspace and productivity setup
- **Books I Read This Month** - Current reading list and reviews
- **And more...** - Additional personal sections coming soon

## Getting Started

Run the development server:

```bash
bun dev
# or
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Tech Stack

- **Framework:** Next.js 16
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Runtime:** Bun
